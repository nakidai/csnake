# csnake
Snakegame written in C.  

Controls  
--
- WASD - movement  
- P - Pause

Supported platforms  
--
Was tested on:  
- FreeBSD 13.2  
- Windows 11 (MSVC)  
- Linux 6.5.8 (glibc 2.38-7)  

Building
--
- On \*nix-like systems you can use GNU Make with configure script providing arguments through environment variables or CMake   
- On Windows you can use CMake
